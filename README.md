# react-native-sunmi-printer

商米内置打印

## Installation

```sh
npm install react-native-sunmi-printer
```

## Usage

```js
import SunmiPrinter from "react-native-sunmi-printer";

// ...

const result = await SunmiPrinter.multiply(3, 7);
```
